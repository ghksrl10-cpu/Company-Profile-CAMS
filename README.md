## 🎨 디자인 시스템

- **컬러**: Navy `#1A3F73` · Red `#D72332` · Off-white `#FAFAFA` · Ink `#0E1A2E`
- **타이포그래피**: Pretendard (한글) + Inter (영문) + JetBrains Mono (숫자)
- **포맷**: 4:3 (1920 × 1440px) · HTML5 단일 파일
- **모션**: 슬라이드 진입 시 fade-up · count-up · bar-grow · line-draw 자동 트리거

## 🖱️ 사용 방법

### 키보드
- **← / →** : 이전 / 다음 슬라이드
- **숫자 키 1–9** : 해당 슬라이드로 바로 이동
- **R** : 첫 슬라이드로 리셋
- **Home / End** : 처음 / 마지막 슬라이드
- **Esc** : 썸네일 레일 토글

### 마우스
- **상단 트래커 클릭** : 해당 섹션으로 이동
- **목차 페이지 타일 클릭** : 해당 슬라이드로 이동
- **도면 핀 호버** : 빨강 펄스 + 한글 라벨 확대
- **카드 호버** : 떠오름 + 빨강 강조
